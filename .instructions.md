# Vue Calculator Project Instructions

## Coding Style and Structure
- Keep code clean, professional, and well-organized.
- Atomize components: break down UI and logic into small, reusable Vue components whenever possible and efficient.
- Avoid placing all code in a single file; prefer modularization.

## Technology Stack
- Use Vue.js (composition API preferred unless otherwise specified).
- Use Ant Design Vue 4 for UI components.
- Use Tailwind CSS 4 for styling.

## Component Design
- Favor functional, focused components (single responsibility principle).
- Reuse components where possible.
- Maintain clear separation of concerns (UI, logic, styles).

## Project Goal
- Build a calculator as a reusable Vue component/library, suitable for publishing to npm and integrating into other projects.

## Example Prompts
- "Refactor this logic into a separate Vue component."
- "Atomize the calculator UI into input, display, and button components."
- "Ensure all styling uses Tailwind 4 and Ant Design Vue 4."
